# Proyecto DevOps

Este proyecto es una práctica de control de versiones utilizando Git y GitHub.

## Objetivo
Simular un flujo básico DevOps

## Estructura del proyecto
mi-proyecto-devops/
│── src/  
│   ├── index.html  
│   ├── styles.css  
│── scripts/  
│   └── deploy.sh  
│── README.md  
│── .gitignore  

## Flujo de trabajo
1. Creación de repositorio
2. Commits en main
3. Creación de ramas con nuevas características
4. Simulación de despliegue con un script

## Comandos Git usados
- git add
- git commit
- git push
- git pull
- git checkout -b
